# streamdownload
Download file using stream.

## Example 
```python
from streamdownload import downloadFile
downloadFile('https://images.pexels.com/photos/2387869/pexels-photo-2387869.jpeg', local='./downloadedfile.png', progress=True)
```

## Requirements
```
Python 3.6
Requests
```
