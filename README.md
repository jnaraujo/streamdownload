# streamdownload
Download file using stream.

## Example 
```python
from streamdownload import downloadFile
downloadFile('https://images.pexels.com/photos/2387869/pexels-photo-2387869.jpeg', savepath='./downloadedfile.png', progress=True)
```

## Requirements
```
Python 3.6
requests==2.22.0
progress==1.5
```
